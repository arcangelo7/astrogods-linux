# AstroGods Linux releases

Binary releases of [AstroGods](https://astrogods.it) for Linux.

AstroGods is a modern, beautifully designed Astrology app focused on self-knowledge. It provides personalized birth chart interpretations, synastry analysis for relationships, and transit readings.

## Installation

## Flatpak

AstroGods has been submitted to [Flathub](https://flathub.org) and is pending review. You can track the submission progress in [PR #7474](https://github.com/flathub/flathub/pull/7474).

### Manual installation

Download the latest `linux-x64-*.tar.gz` from [Releases](https://github.com/arcangelo7/astrogods-linux/releases), extract and run:

```bash
tar -xzf linux-x64-*.tar.gz
cd linux-x64-*/bundle
./astrogods
```
