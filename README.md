## Installation

### Flatpak (recommended)

AstroGods is available on [Flathub](https://flathub.org/apps/it.astrogods.app):

```bash
flatpak install flathub it.astrogods.app
```

### Manual installation

Download the latest `linux-x64-*.tar.gz` from [Releases](https://github.com/arcangelo7/astrogods-linux/releases), extract and run:

```bash
tar -xzf linux-x64-*.tar.gz
cd linux-x64-*/
./astrogods
```

## Requirements

- Linux x86_64
- GTK 3.0
- OpenGL support
